# 🤖 Smart Loan Assistant Bot

Smart Loan Assistant Bot is a Telegram bot that helps users calculate loan details such as EMI, yearly interest, and total payable amount based on selected bank and loan type.

🔗 **Use the Bot:**  
https://t.me/SmartLoanAssistantBot

---

## 🚀 Features

- Select from multiple banks
- Choose different loan types
- Calculate Monthly EMI
- Calculate Yearly Interest
- Calculate Total Payable Amount
- Restart calculation instantly
- Webhook-based deployment (FastAPI)

---

## 🛠 Tech Stack

- Python 3
- FastAPI
- python-telegram-bot (v20+)
- Render (Cloud Deployment)
